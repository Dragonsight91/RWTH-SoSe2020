# Artificial plants

Drawing Program knows the Following instructions

- `F` draw short line
- `-` turn right
- `+` turn left
- `[` save current position
- `]` go back to last saved position

`F[+FF][--F]F[]+F[+FF][--F]FF[+FF][--F]F][--F[+FF][--F]F]F[+FF][--F]F`

Start with `F` and use `F -> F[+FF][--F]F` for n-times
